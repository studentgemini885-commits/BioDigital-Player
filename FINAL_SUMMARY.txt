🔧 দ্রুত কমান্ড রেফারেন্স:
═══════════════════════════════════════════════════════════════════

🐛 সাধারণ সমস্যা সমাধান:
═══════════════════════════════════════════════════════════════════

Q: Build খুব ধীর?
A: স্বাভাবিক! প্রথমবার ডিপেন্ডেন্সি ডাউনলোড করে। 
   আপেক্ষা করুন - 15-25 মিনিট সময় নিতে পারে।

Q: "ANDROID_HOME not set" এরর?
A: পরিবেশ ভেরিয়েবল সেট করুন:
   - Windows: set ANDROID_HOME=C:\path\to\Android\Sdk
   - Mac/Linux: export ANDROID_HOME=$HOME/Android/Sdk

Q: Build ব্যর্থ হচ্ছে?
A: Gradle ক্লিন করুন:
   cd android && ./gradlew clean && ./gradlew assembleRelease

Q: Java সংস্করণ ত্রুটি?
A: Java 21 ব্যবহার করুন:
   java -version (চেক করুন)
   JAVA_HOME সেট করুন
