🚀 ৩ ধাপে APK তৈরি করুন:
═══════════════════════════════════════════════════════════════

ধাপ 1: আপনার মেশিনে ইনস্টল করুন
────────────────────────────────────
Windows/Mac/Linux এ:
• Java 21 ডাউনলোড করুন (https://bit.ly/java21)
• Android Studio ডাউনলোড করুন (https://bit.ly/android-studio)
• Node.js 18+ ডাউনলোড করুন (https://nodejs.org/)

বিস্তারিত: দেখুন SETUP_INSTRUCTIONS.md


ধাপ 2: কমান্ড চালান
────────────────────
টার্মিনাল খুলুন এবং:

npm install
npx expo prebuild --clean
cd android
./gradlew assembleRelease

(প্রথম বার 15-25 মিনিট লাগবে)


ধাপ 3: APK পান
────────────────
এখানে থাকবে:
android/app/build/outputs/apk/release/app-release.apk

ইনস্টল করুন:
• ফোনে কপি করে ট্যাপ করুন, অথবা
• আপনার ক্রেডেনশিয়ালস দিয়ে ADB করুন

❓ সাধারণ প্রশ্ন:
═══════════════════════════════════════════════════════════════

Q: Build খুব ধীর কেন?
A: স্বাভাবিক। প্রথমবার সব ডিপেন্ডেন্সি ডাউনলোড করে (~3GB)
   পরবর্তী বার দ্রুত হবে।

Q: "ANDROID_HOME not set" এরর?
A: সেটআপ করুন: SETUP_INSTRUCTIONS.md এ দেখুন

Q: Windows এ কমান্ড কাজ করছে না?
A: PowerShell ব্যবহার করুন বা WSL

Q: Build ব্যর্থ হয়েছে?
A: ./gradlew clean চালান, তারপর আবার শুরু করুন
